## RoPro - Comments - Post Comment

URL: https://api.ropro.io/postComment.php \
Method: POST \
Auth Required: Yes

## Request Body Parameters

`itemid` - integer - The item the comment is for\
`userid` - integer - Your Roblox UID\
`comment` - string - Your actual comment
