## RoPro - Invites - Create Invite

URL: https://api.ropro.io/createInvite.php \
Method: POST \
Auth Required: No

## Query Parameters

`universeid` - string - The universe the invite is for\
`serverid` - string - The Job ID the invite is for

---

_Example Response_

```
https://ropro.io/join/ABC123
```
